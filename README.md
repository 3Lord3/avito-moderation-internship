# Тестовое задание Avito Frontend осень 2025
## Установка и запуск
### Docker Compose
```
// Клонирование репозитория
git clone git@github.com:3Lord3/avito-moderation-internship.git
cd avito-moderation-internship

docker compose up --build
```

Приложение будет доступно по адресу: http://localhost:5173

## Обоснование выбора необязательных технологий
**Основа: Библиотека компонентов Shadcn UI**. Выбрана за гибкую настройку - представляет из себя стилизованные компоненты Radix UI с установкой локально только использованных компонентов. Ускоряет разработку и уменьшает количество проблем с дизайном

Подтянутые зависимости Shadcn:
- Tailwind CSS - современная CSS-In-JS библиотека
- Lucide React - библиотека иконок
- Recharts - библиотека для построения графиков
- Embla Carousel - библиотека для галереи изображений
- Некоторые компоненты Radix Primitives - готовые компоненты без стилей

**Язык: TypeScript**. Современный способ писать frontend. Отличается от JS сильной типизацией, важно для надёжности приложения 

**State Managemen: Redux ToolKit**. Выбрана по причине наличия опыта работы с данной библиотекой в прошлом

**API: RTK Query**. Выбран из-за встроенности в библиотеку RTK
